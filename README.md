# flash-open-vscode

Open `.fla` files in Adobe Flash.

## Install

With VS Code

```bash
$ code --install-extension pxgamer.flasho
```

## Usage

Right click on a `.fla` file in the `tree-view` that you want to open in Adobe Flash.
